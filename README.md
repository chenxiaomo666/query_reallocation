# query_reallocation
## 功能
定时查询部分学校的调剂信息发布情况，并发邮件提醒

## 使用说明
### 账户相关信息配置
在config.py文件中填写自己邮箱的相关信息（邮箱服务器地址、账户、密码（授权码））
并填写接收人邮箱列表（代表可以给多人发送邮件）

### 学校相关信息配置
在config.py文件school_url字典中以{"school_name": "url"}填写，url需为该校发布信息主页
